## Component Breakdown

### Components (`src/components/`)

- **MapView.tsx**: Main map visualization component
  - Integrates Leaflet map with OpenStreetMap tiles
  - Renders clinic markers with distance-based colors
  - Displays user location marker
  - Handles marker click interactions

- **FilterPanel.tsx**: Left sidebar filter controls
  - Maximum distance slider (1-20 km)
  - Clinic type checkboxes
  - Specialty/focus checkboxes
  - Availability feature checkboxes

- **ClinicDetails.tsx**: Right sidebar clinic information
  - Contact information display
  - Operating hours and appointment procedures
  - Clinical focus areas
  - Accessibility features
  - Distance from user location

### Hooks (`src/hooks/`)

- **useUserLocation.ts**: Geolocation management
  - Requests user location permission
  - Manages location state and error handling
  - Returns location coordinates and loading state

- **useClinicFilters.ts**: Filter logic and state
  - Manages all filter states
  - Computes filtered clinic list based on active filters
  - Provides filter update functions

### Utilities (`src/utils/`)

- **distance.ts**: Haversine formula implementation
  - Calculates great-circle distance between two coordinates
  - Returns distance in kilometers

- **colorScale.ts**: Visual encoding functions
  - Maps distance to color values
  - Provides marker color based on proximity

### Data (`src/data/`)

- **clinics.ts**: Mock dataset
  - 18 clinic records with complete metadata
  - TypeScript interface definitions
  - Exportable clinic array

## Configuration Files

- **package.json**: NPM dependencies and scripts
- **tsconfig.json**: TypeScript compiler options
- **vite.config.ts**: Vite build tool configuration
- **.eslintrc.cjs**: ESLint linting rules
- **.gitignore**: Git ignore patterns

## Build Output

The `dist/` folder is generated by `npm run build` and contains:
- Optimized JavaScript bundles
- Minified CSS
- Static assets
- Production-ready HTML

This folder is deployed to GitHub Pages via the `gh-pages` branch.
