# PATH — Psychological Access To Help

A modern web application for finding psychological support and safe paths to clinics.

## Features

- Dynamic neural network background animation
- Smooth scroll navigation
- Animated sections with Framer Motion
- Responsive design
- Minimalist newspaper-style design

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The application will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

## Tech Stack

- React 18
- Vite
- Framer Motion (for animations)
- CSS3
